showFlix API

This app is to provide information about all kinds of movies, their genres and directors! This is the one stop shop for movie lovers who want to learn about or keep track of movies they adore by creating a list of their favorites! All this data about these different movies will be brought by this myFlix API.

Request URL HTTP Method Response
Return a list of ALL movies to the user /movies GET A JSON object holding data about the list of all movies. Example: { title: '13 Hours: The Secret Soldiers of Benghazi', genre: 'Action' director: 'Michael Bay' }, { title: 'Lord of the Rings: The Fellowship of the Ring', genre: 'Fantasy' director: 'Peter Jackson' }
Return data about a single movie by title to the user /movies/:Title GET A JSON object holding data about a single movie. Example: { "title": "13 Hours", "description": "sample-description", "genre": "Action", "director": "Michael Bay", "id": "1" }
Return data about a genre by name/title /genres/:Name GET A JSON object holding data about a genre by name
Return data about a director (bio, birth year, death year) by name /directors/:Name GET A JSON object holding data about directors. Example: { "name": "Michael Bay", "bio": "bio-description", "birth": "0000", "death": "0000", }
Allow new users to register /user/register POST A JSON object holding data containing user information. Example: { "Username": "kylayouu", "Password": "123456", "Email": "abc@gmail.com", "FavoriteMovies": [], }
Allow users to update their user info /users/:username/edit_profile PUT A JSON object with updated user info. Example: { "Username": "kylayouu", "Password": "123456", "Email": "abc@gmail.com", "FavoriteMovies": [] }
Allow users to add a movie to their list of favorites /users/:username/favorites POST A JSON object with the updated list of favorites for a user. Example: { "Username": "kylayouu", "Password": "123456", "Email": "abc@gmail.com", "FavoriteMovies": [ "id:1", "id:2" ] }
Allow users to remove a movie from their list of favorites /users/:username/favorites DELETE A JSON object with the updated list of favorites for a user. Example: { "Username": "kylayouu", "Password": "123456", "Email": "abc@gmail.com", "FavoriteMovies": [ "id:1" ] }
Allow existing users to deregister /users/:username DELETE Text displaying user has been removed. Example: "kylayouu was successfully deleted."
